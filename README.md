# Santhosh Dhandapani - Portfolio Website

A modern, animated portfolio website showcasing the software development expertise. Built with React, Framer Motion, and Aceternity UI components with a focus on performance, accessibility, and stunning visual effects.

## 🚀 Features

### ✨ **Modern Animations & Effects**
- Smooth page transitions with Framer Motion
- Interactive scroll-based animations
- Particle background effects with BackgroundBeams
- Tracing beam animations for timeline sections
- Hover effects with blue glow and scaling

### 🎨 **Professional Design**
- Clean, dark theme with blue accent colors
- Responsive grid layouts (no horizontal scrolling)
- Consistent spacing and typography
- Modern card-based components
- Active navigation highlighting

### 📱 **Responsive & Accessible**
- Mobile-first responsive design
- Accessible navigation with keyboard support
- Semantic HTML structure
- Optimized for all screen sizes

### 🎯 **Interactive Components**
- Dynamic navigation with active section detection
- Technology tooltips with proficiency levels
- External link components with proper security
- Reusable UI components (Cards, Grids, Icons)

## 🚀 Getting Started

### Prerequisites
- Node.js 16+ 
- npm or yarn

### Installation
```bash
# Clone the repository
git clone https://github.com/santhoshdhandapani/portfolio

# Navigate to project directory
cd santhosh-portfolio

# Install dependencies
npm install

# Start development server
npm start
```

### Build for Production
```bash
# Create production build
npm run build

# Serve production build locally
npm run serve
```

## 🎨 Design System

### **Color Palette**
- Primary: Blue (#3B82F6, #60A5FA)
- Background: Black (#000000)
- Cards: Gray-900 (#111827)
- Text: White (#FFFFFF), Gray-300 (#D1D5DB)

### **Typography**
- Headings: Bold, responsive sizing
- Body: Clean, readable fonts
- Code: Monospace for technical content

### **Spacing**
- Consistent `mb-16` (4rem) between sections
- Responsive padding and margins
- Grid gaps optimized for different screen sizes

## 📱 Responsive Breakpoints

- **Mobile**: < 768px (1 column layouts)
- **Tablet**: 768px - 1024px (2 column layouts)
- **Desktop**: > 1024px (3+ column layouts)
